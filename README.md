🎓 School Manager (Java)

Simple console-based School Manager system built with Java.

✨ Features

- Add Student
- View All Students
- Search Student by ID
- Delete Student
- Simple CLI menu

---

📁 Project Structure

school-manager/
 ├── Main.java
 ├── Student.java
 └── SchoolManager.java

---

⚙️ Requirements

- Java JDK 8 or higher

Check version:

java -version

---

▶️ How to Run

javac *.java
java Main

---

🧠 How It Works

- "Student.java" → Model (student data)
- "SchoolManager.java" → Logic (CRUD operations)
- "Main.java" → User interface (menu + input)
